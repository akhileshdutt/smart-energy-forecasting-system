<h1 align="center">⚡ Smart Energy Consumption Forecasting System</h1>

<p align="center">
A Machine Learning-based system to predict energy consumption, estimate electricity cost, detect peak usage, and provide optimization recommendations.
</p>

<hr>

<h2>📌 Problem Statement</h2>
<p>
Most households only receive monthly electricity bills without insights into usage patterns, future consumption, or cost optimization.
</p>

<h2>💡 Solution</h2>
<p>
This system forecasts future energy consumption using Machine Learning and converts predictions into actionable insights like cost estimation, peak usage detection, and recommendations.
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
<li>📊 Synthetic Data Generation</li>
<li>⚙️ Feature Engineering with Lag Variables</li>
<li>🤖 Machine Learning Model (Random Forest)</li>
<li>📈 Energy Forecasting (Next 30 Days)</li>
<li>💰 Cost Estimation</li>
<li>⏰ Peak Usage Detection</li>
<li>💡 Recommendation Engine</li>
<li>🖥️ Streamlit Dashboard</li>
</ul>

<hr>

<h2>🛠️ Tech Stack</h2>
<ul>
<li>Python</li>
<li>Pandas, NumPy</li>
<li>Scikit-learn</li>
<li>Streamlit</li>
</ul>

<hr>

<h2>▶️ How to Run</h2>

<pre>
pip install -r requirements.txt
streamlit run app.py
</pre>

<hr>

<h2>📊 Output</h2>
<ul>
<li>Predicted energy consumption</li>
<li>Estimated electricity bill</li>
<li>Peak usage hours</li>
<li>Optimization recommendations</li>
</ul>
